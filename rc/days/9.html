Day 8 - Bugs Be Gone

Today I learned...

  • indexof() vs findindex() are two JS array methods to locate elements within any array; however, the latter allows you to pass a callback function to execute on each value in the array, well suited for locating the index of a specific property within an array of objects. e.g. Arr.findIndex(x => x.property === value);
  • "Lifting state up" and useContext are two ways in React to send data upwards from a child component into the parent component. 'Lifting state up' requires passing the updater function from the parent via props to the child, and updating the value within the child component... and useContext I'll need to dig into that syntax.
  • Controlled input is a React concept in which form input data is handled by state and dynamically updated via the onChange event listener.
