Day 13 - Discusssions, Dev Tools, & DSA

Today I learned...

  • A Terminal User Interface (TUI) is a type of user interface that uses text and simple graphical elements, often including menus, forms, and navigational components that make it easier to interface with as compared to a traditional CLI; it differs from a GUI in that a TUI is less complex in terms of design, accessibility, and interaction. Vim & ranger are examples of TUIs.
  • jq is a command-line tool that is designed to query, filter, and manipulate JSON data. curl is a command-line tool used to transfer data to or from a server and is widely used for interacting with web services and APIs. Combining curl with jq, a user can fetch data using an API and use jq to process it for reporting/scripting tasks.
  • Dual Boot, Virtual Machines, and Docker are different ways of managing/running operating systems or environments on your computer. Dual booting involes installing multiple operating systems on the same physical machine, Virtual Machines is a emulation of a physical computer which runs its own operating system and applications, and Docker is a containerization platform that runs containers on the host operating system's kernel but provided isolated environments for applications.
