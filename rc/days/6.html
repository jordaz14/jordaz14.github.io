Day 6 - dnd kit

Today I learned...

  • The JavaScript map() method has takes three parameters map((element, index, array)); if you'd like to capture the index of an element via map without utilizing the mandatory element parameter in your code, you can insert '_' in its place to indicate that it's not being used.
  • At a high level, dnd kit is a React library for building drag & drop interfaces. If you'd like to create a sortable drag & drop list, you'll need to: wrap your elements in both a DndContext & SortableContext, map through your existing list to output a SortableItem element, and create a handleDragEnd function to swap the index of two elements once a user finishes dragging an element.
  • Object destructuring allows you to extract multiple properties from an object in a single statement and bind them to variables. Array destructuring is similar in that it allows you to unpact values from arrays without repeating the variable = arr[idx] syntax for each value.
