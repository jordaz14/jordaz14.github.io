Day 36 - dns, net, and tls in Node.js

Today I learned...

  • dns, net, and tls are core modules in Node.js that provide functionality for network-related tasks. The dns module allows you to perform dns lookups, resolves, and reverses; the net module allows you to create and start TCP servers & clients; the tls module provides TLS functionality, allowing you to create TLS servers/clients and interact with secure connections.
  • Middleware functions in express are functions that have access to request and response objects before and after their route handling. Each function takes a req, res, and next parameter, with the latter being a function that dictates control flow by passing control the next middleware function.
  • The Performance.now() method call provides a time stamp with precision up to microseconds relative to start of a page load, whereas the Date.now() call is based on the system clock and is measured in milliseconds.
