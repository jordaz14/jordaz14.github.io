Day 28 - Back to Backend

Today I learned...

  • Traditional server hosting entails the management of a physical server which runs your applications, requiring you to configure the entire server environment, including the OS, web server, and application itself. Serverless Functions (i.e. Functions as a Server) is a cloud computing model where the infrastructure is abstracted away and the developer can write code that runs in response to events.
  • A PostgreSQL client is a tool which interfaces with a PostgreSQL database, which allows you to connect to the db server, execute SQL queries, manage tables, and handle other admin tasks. Different types of clients include command-line clients, GUI clients, Library clients, and Web-based clients. Popular library clients for node.js include pg & postgres.
  • Nodemon is a tool for Node.js which automatically restarts your server file when changes are detected (i.e. hot refresh). The configuration file includes specifying which files to watch, which to restart, which to ignore, and how to execute the command.
