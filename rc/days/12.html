Day 12 - Back to Business

Today I learned...

  • There are three ways to cache data in web applications: LocalStorage, Session Storage, and Cookies; LocalStorage typically holds around 5-10 MB per origin with data persisting after the browser is closed; SessionStorage is similar in storage capacity but only persists for as long as a page remains open; and, Cookies are typically limited to 4KB and used for session management.
  • Embeddings are representations of objects, such as words, sentences, or items, in a continuous vector space. Embeddings are used in machine learning and natural language processing (NLP) to represent data in a way that captures semantic relationships and similarities between objects.
  • Lazy initialization is the tactic of delaying the creation of an object or calculation of a value until the first time it is needed; this can be used in React's useState() such that you pass a function into useState instead of a direct value so that it will only be excuted once during the initial render to compute the initial value. The appropriate syntax would be: useState(() => expensiveCalculation());
