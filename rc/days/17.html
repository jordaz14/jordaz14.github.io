Day 17 - Demo Day

Today I learned...

  • Folk computing refers to the informal methods and practices people use to solve problems with computing technology, and it often involves self-learning, practical solutions, and community sharing; an example of this would include custom scripting with Python or JavaScipt.
  • Lisp is the oldest high-level programming language which derives its name from 'LISt Processing"; Lisp programs and data are represented as lists, which allows you to treat code as data and vice versa.
  • JavaScript's native Promise object does not support cancellation directly, so additional logic is often implemented to handle cancellation.
