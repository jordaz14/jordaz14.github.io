Day 34 - Measuring Connection Times

Today I learned...

  • Chrome Performance API is a set of web APIs supported by modern browsers that developers can use to measure the performance of their web apps. Features of the Performance API include timestamps, navigation timing, resource timing, and paint timing. More specifically, PerformanceNavigationTiming can be used for measuring DNS & TCP connection times.
  • There are three versions of HTTP: 1.0, 1.1, 2, and 3. With regard to connection handling, 1.0 creates a new TCP for each request; 1.1 allows multiple requests and responses to be sent over a single TCP connection; 2 allows multiple for multiple requests being sent over a single connection simultaneously, avoiding blocking issues; 3 uses the QUIC transport protocol, based on UDP.
  • A HTTP Archive (HAR) file is a JSON-formatted text file used to record the network activity of a web browser, and it contains request/response, timing, and resource information.
