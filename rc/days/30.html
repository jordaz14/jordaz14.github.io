Today I learned...

  • Concurrency refers to the simultaneous execution of multiple tasks/operations, allowing them to make progress independently of one another. However, concurrency in JavaScipt differs slightly since it operates on a single thread & pertains more to managing the order of execution in a way that maximizes efficiency (i.e. using async operations to queue to the event loop).
  • A testing framework is a set of tools that provide a structured way to execute tests for code. Key components of testing frameworks include a test runner, assertion library, test fixtures, test suites, and code coverage. Jest is a testing framework for JavaScript.
  • websocket is a protocol which allows a server and client to send messages to each other simultaneously, and Socket.IO is a library which provides an abstraction over WebSocket & offers features such as event handling, rooms, and reconnection..
