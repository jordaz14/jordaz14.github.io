Day 26 - Traffic Trouble: Refactored & Revamped

Today I learned...

  • A REPL, Read-Eval-Print-Loop, is an interactive programming environment that allows for writing and executing code in real time. Node.js provides a REPL environment by running 'node' in the terminal.
  • First-class and higher-order functions both belong to the functional programming paradigm. First-class functions are those which are treated like any other variable and can be passed as arguments to other functions, returned as values, and assign to variables. A higher order function is a function that takes other functions as arguments or returns a function.
  • Function currying is a technique where a function that takes multiple arguments is transformed into a series of functions, eaching taking one argument.
