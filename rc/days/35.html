Day 35 - Puppeteer and Puppeteer-har

Today I learned...

  • Puppeteer is a library developed by Google that offers a high-level API to control headless Chrome/Chromium browsers programmatically, often used for web scraping, automated testing, performance monitoring, and generating PDFs/screenshots of web pages. Puppeteer-har is a package that integrates with Puppeteer to allow developers to capture HAR files.
  • A type declaration file (.d.ts) in TypeScript is used for integrating JavaScript libraries and modules by describing the shape and types of the JS code. DefinitelyTyped is a repo which provides type definitions for popular libraries and can be installed via 'npm install --save-dev @types/'
  • A userland package in npm is any package that is not part of the core Node.js runtime and instead is maintained by community developers. Whereas Node has core modules such as fs, http, and path, popular userland packages include express and axios. Additionally, 'core modules' differs from 'standard libraries' conceptually in that modules refer to a runtime environment, whereas a standard library refers a programming language at-large.
