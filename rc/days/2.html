Today I learned...

  • You can split & rename terminals in VSCode to easily toggle between different directories... it's the little things.
  • NPX differs from NPM in that it allows you to execute packages without installing globally. I'd like to see some use cases to get a better sense though.
  • Vercel isn't only for hosting Next.JS projects, so it seems like the best service for my project. Although I'm still not quite sure why some recommend Heroku (backend) + Netlify (frontend).

Today featured a pair programming workshop. Although I did not participate today (out of nerves, candidly), I recognize it'll be an important part of my growth. (1) For trying new things, & (2) better articulating my own code and working collaboratively with others.
