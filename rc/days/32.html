Day 32 - The MVC Architecture Pattern

Today I learned...

  • Object-Relational Mapping (ORMs) is a programming technique to interact with a database using the object-oriented paradigm, as opposed to writing raw SQL queries. More specifically, tables are mapped to objects and allow you to abstract SQL into JavaScript. Examples of ORMs in JS include Sequelize, Mongoose, and TypeORM.
  • A software architecture pattern is a high-level design pattern that defines the structure of a software system. Common architecture patterns include layered architecture, microservices architecture, client-server architecture, and model-view-controller (MVC). The Model-View-Controller pattern separates the application into the model (data and business logic), view (UI), and controller (user input and interactions)
  • The TCP handshake refers to the process of creating a connection between a client and server over TCP. This process is divided into three steps: (1) Syn - client sends a packet with a synchronize flag to the server; (2) Syn-Ack - server responds with a SYN & ACK flag; (3) Ack - client sends a packet with the ACK flag set and a reliable connection is established.
